# Reservation Coding Assignment

Front end heroku link [here](https://takehome-usv.herokuapp.com/).

## Features

Users are able to:

- Add Restaurants
- Search Restaurants
- Create and Update Reservations
- List all Reservations
- View Details for Individual Reservations or Restaurants

Responsive

- Mobile (320px - 480px)
- Desktop
